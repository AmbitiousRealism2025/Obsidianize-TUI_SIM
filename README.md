# Bun Vibecoder Starter

A streamlined starter project for building high-performance applications with Bun.js and TypeScript. Designed for a "vibecoding" workflow that prioritizes speed, an excellent developer experience (DX), and minimal configuration.

## Available Commands

- **`bun dev`**: Runs the development server with hot-reloading.
- **`bun start`**: Runs the application.
- **`bun test`**: Executes the test suite.
- **`bun build`**: Creates a production-ready build of the application in the `./build` directory.

## Getting Started

1. Install dependencies:
   ```bash
   bun install
   ```

2. Run the development server:
   ```bash
   bun dev
   ```

The server will be available at `http://localhost:3000`.
